{%- comment -%} Video cards grid - displays video thumbnails with click-to-play. Supports both YouTube video IDs and custom iframe URLs (starting with "http"). YouTube videos show optimized thumbnails; custom URLs show a placeholder. Uses eleventy-img to fetch and optimize YouTube thumbnails for fast loading. Videos only load when clicked, saving bandwidth until user interaction. Parameters: - videos: Array of video objects with properties: - id: YouTube video ID or custom iframe URL (required) - title: Video title (required) - thumbnail_url: Computed property - YouTube thumbnail URL or null for custom URLs - reveal: If true, adds data-reveal animation (default: true) - heading_level: Heading level for titles, defaults to 3 (h3) - expand: If true, videos fill available width (1=100%, 2=50%, 3+=33.3%) instead of the standard card grid layout (default: false) Usage: {% include "design-system/video-cards.html", videos: page.videos %} {% include "design-system/video-cards.html", videos: page.videos, expand: true %} Where page.videos = [ { id: "dQw4w9WgXcQ", title: "YouTube Video" }, { id: "https://player.vimeo.com/video/123456", title: "Vimeo Video" }, ... ] {%- endcomment -%} {%- assign reveal = reveal | default: true -%} {%- assign heading_level = heading_level | default: 3 -%}
